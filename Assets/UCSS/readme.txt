
***

WWW wrapper for Unity

***

WWW wrapper – simple but usable add-on for Unity3D WWW class. 
Allows to make requests using one line of code and get results of request via callback.
WWW wrapper adds customizable timeouts to Unity3D WWW class functionality.
Using WWW wrapper your code will be clean and easy to understand.

Unity 3.4.0+ compatible

Please, read full documentation on plugin website http://ucss.webjema.com/http/

Check tests scene in [UCSS/Tests] folder.

***

AMF for Unity is part of Client-Server Solution for Unity http://ucss.webjema.com/
